环境安装

dlv安装

1

$ go install github.com/go-delve/delve/cmd/dlv@latest

go 版本小于1.16的用下面方式安装

1
2
3

$ git clone https://github.com/go-delve/delve
$ cd delve
$ go install github.com/go-delve/delve/cmd/dlv

如果已构建成二进制可执行文件，我们可以使用 dlv exec 命令开始调试：

1

$ dlv exec /path/to/go_binary_file

对于已经运行的程序，可以使用 attach 命令，进行跟踪调试指定 pid 的 Go 应用：

1

$ dlv attach pid

对于 go 源码文件，可以用 debug 调试

1

$ dlv debug main.go

对于需要命令行参数才能启动的程序，我们可以通过 – 来传递命令行参数，比如如下:

1
2

$ dlv debug github.com/me/foo/cmd/foo -- -arg1 value
$ dlv exec /mypath/binary -- --config=config.toml

Viewing program variables and memory:
    args ----------------- Print function arguments.
    display -------------- Print value of an expression every time the program stops.
    examinemem (alias: x)  Examine raw memory at the given address.
    locals --------------- Print local variables.
    print (alias: p) ----- Evaluate an expression.

Type help followed by a command for full documentation.

package cal

type Cal struct {
        X1 int
        X2 int
}

func (c *Cal)Add() int {
        return c.X1 +c.X2
}
func (c *Cal)Sub() int {
        return c.X1 -c.X2
}

func (c *Cal)Mul() int {
        return c.X1 *c.X2
}

func (c *Cal)Div() int {
        return c.X1 / c.X2
}

基础使用

常用选项

  • c（continue）继续运行程序，直到结束或者遇到断点
  • r（restart）重启程序
  • s（step）单步进入
  • si（step-instruction） 单步汇编指令（很少用到）
  • so（stepout）跳出当前函数
